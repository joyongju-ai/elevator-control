# elevator-control

# STM32 기반 엘리베이터 제어 시스템

## 프로젝트 개요
STM32G474 마이크로컨트롤러를 활용한 5층 엘리베이터 제어 시스템입니다. 하드웨어 담당 1명과 팀을 구성해 개발했으며, 소프트웨어 전반을 담당했습니다.

## 개발 환경
- MCU : STM32G474
- IDE : STM32CubeIDE
- 언어 : C

## 주요 기능
- 홀 센서 기반 층 감지 (자석 자기장 활용)
- TIM3 PWM을 통한 모터 방향 제어
- 키패드 입력 처리 및 다중 층 호출 큐 관리
- 7-세그먼트 현재 층 표시
- 도어 열림 상태에서 주행 차단 인터락 로직
- HAL_GetTick 기반 논블로킹 구조

## 핵심 설계 원칙
에러 상황에서도 승객이 위협을 느끼지 않도록 안정적인 정지를 최우선 설계 원칙으로 삼았습니다.
